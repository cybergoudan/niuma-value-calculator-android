# 牛马价值计算器（Android）

自动统计你在各个 App 上花的时间，并按你设定/计算的「时间价值（元/小时）」换算成**娱乐成本**。

## 下载 APK
进入 Releases 下载最新的 `*-debug` 版本：
https://github.com/cybergoudan/niuma-value-calculator-android/releases

## 关键能力（自动挡）
- 申请「使用情况访问权限（Usage Access）」后：
  - 自动统计抖音/快手/B站/微博等 App 的使用时长
  - 按类别（工作/学习/娱乐）汇总
  - 超过阈值提醒：例如连续刷 2 小时 → 提示“你刚刚烧掉了 XX 元”

## 本地编译
Android Studio + JDK 17
